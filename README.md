### 使用方法：
    从git仓库下载之后，命令行运行npm install，下载相应资源包之后可选择通过网页打开、electron在线预览或者打包为桌面应用都可以。
    直接打开index.html可以网页预览
    命令行运行npm run build可以通过electron预览
    命令行运行npm run packager可以打包为windows桌面应用（暂时只限于windows系统，其他系统暂不支持）
